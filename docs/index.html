Installation

Prerequisites

Python 3.8 or higher is required. Google Chrome browser is needed for web scraping functionality.

Quick Install

git clone https://github.com/LimJih00n/copernicus-marine-toolkit.git
cd copernicus-marine-toolkit
pip install -r requirements.txt

Verify Installation

python test_installation.py

This will run comprehensive tests on all dependencies and generate a test report.

Usage Examples

Data Collection

# Basic scraping
python scrape_copernicus.py

# Enhanced scraping with parallel downloads
python scrape_copernicus_enhanced.py --workers 5

# Disable cache for fresh downloads
python scrape_copernicus_enhanced.py --no-cache

Data Analysis

import copernicus_utils as cu

# Load NetCDF dataset
ds = cu.load_dataset('data.nc')

# Extract region (East Sea)
east_sea = cu.subset_region(ds, lon_range=(120, 135), lat_range=(30, 45))

# Calculate spatial mean with latitude weighting
mean_sst = cu.calculate_spatial_mean(east_sea['temperature'])

# Detect marine heatwaves
mhw_events = cu.detect_marine_heatwaves(east_sea['temperature'])
